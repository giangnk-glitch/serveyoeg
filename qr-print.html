QR Codes — Cyber Hub Survey
Tạo và in mã QR cho 6 cơ sở. Khách quét → điền khảo sát → tự động ghi vào Google Sheets.
Hướng dẫn deploy lên GitHub Pages
1
Tạo repository GitHub tên cyberhub-survey (public), upload 3 file: survey.html, dashboard.html, qr-print.html
2
Vào Settings → Pages, chọn source = main branch / root. GitHub tự cấp URL dạng https://username.github.io/cyberhub-survey/
3
Vào script.google.com, tạo project mới, paste nội dung file Code.gs, thay SHEET_ID bằng ID Google Sheet của anh
4
Deploy Apps Script: Deploy → New deployment → Web App. Copy URL → paste vào SCRIPT_URL trong cả survey.html và dashboard.html
5
Quay lại trang này, nhập URL GitHub Pages của anh → bấm Tạo QR → in và dán tại các cơ sở
